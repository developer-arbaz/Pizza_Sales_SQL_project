# 🍕 Pizza Sales Data Analysis using SQL

## 📌 Project Overview

This project focuses on analyzing pizza sales data using SQL to extract meaningful insights and support data-driven decision-making. The analysis covers order trends, revenue generation, customer preferences, and product performance.

The goal of this project is to demonstrate strong SQL skills, data analysis thinking, and the ability to derive business insights from raw data.

---

## 🛠️ Tools & Technologies Used

* SQL (MySQL)
* Pizza Sales Dataset
* Concepts: Joins, Aggregations, Window Functions, Subqueries

---

## 📂 Database Schema

The project uses the following tables:

* **orders** – Stores order date and time
* **order_details** – Contains quantity and pizza IDs
* **pizzas** – Includes pizza size and price
* **pizza_types** – Contains pizza name and category

---

## 📊 Key Business Questions Solved

### 🔹 Basic Analysis

* Retrieve the total number of orders placed
* Calculate total revenue generated
* Identify the highest-priced pizza
* Find the most common pizza size

### 🔹 Intermediate Analysis

* Top 5 most ordered pizza types
* Category-wise quantity distribution
* Distribution of orders by hour
* Category-wise pizza count

### 🔹 Advanced Analysis

* Average pizzas ordered per day
* Top 3 pizzas based on revenue
* Percentage contribution of each category to revenue
* Cumulative revenue over time
* Top 3 pizzas per category using window functions

---

## 📈 Key Insights

* Identified top-selling pizzas contributing the most revenue
* Found peak ordering hours for better staffing decisions
* Analyzed category performance for inventory planning
* Understood customer preferences and ordering patterns
